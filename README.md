# Laravel Postman

## Installation

Install the package:

```shell
composer require akkurateio/laravel-postman
```

Publish the config file:

```php
php artisan vendor:publish --provider="Akkurateio\LaravelPostman\LaravelPostmanServiceProvider" --tag="swagger-config"
```
